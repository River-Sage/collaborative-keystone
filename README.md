# Collaborative Keystone

Collaborative Keystone is an open source platform for democratic issue identification, solution development, and implementation tracking.

## Mission

Create a durable public process for surfacing urgent problems and coordinating practical action.

## Version 1 Scope

- World locale first
- Issue Board
- Solution Board
- Monthly cycle
- Public archive
- Open governance groundwork

## Core Principles

- Open by design
- Transparent rules
- Auditable moderation
- Practical solutions with completion criteria
- Built to be forked for other locales

## Project Status

Planning and specification phase.
