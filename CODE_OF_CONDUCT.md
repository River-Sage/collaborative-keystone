# Code of Conduct

Collaborative Keystone is a serious public-interest project.

We want contributors who are direct, thoughtful, and capable of disagreement without sabotage.

## Expected Behavior

- be constructive
- be honest
- be clear
- criticize ideas without demeaning people
- point out flaws directly
- document important reasoning when changing core behavior
- respect the public mission of the project

## Unacceptable Behavior

- harassment
- threats
- doxxing
- hate-driven abuse
- deliberate disruption
- intentional deception
- bad-faith manipulation of discussion or contribution processes
- attempts to hide important governance or moderation changes

## Scope

This code of conduct applies to:

- repository discussions
- issues
- pull requests
- governance discussion spaces
- project communication channels

## Enforcement

Maintainers may remove, restrict, or reject participation that undermines the project or the people working on it.

Where practical, major enforcement actions should be documented.
